## 🚀 Install Lua

### Ubuntu / Debian

```bash
sudo apt update
sudo apt install lua5.4
```

### RHEL / Rocky Linux

```bash
sudo dnf install lua
```

### Arch Linux

```bash
sudo pacman -S lua
```

### Verify Installation

```bash
lua -v
```
